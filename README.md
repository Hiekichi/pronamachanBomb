# pronamachanBomb
プロ生ちゃんが爆弾を使って敵を退治するゲームっぽい何か

動かし方
　jupyter notebookのノート形式でソースコードが保管されていますので、
　ダウンロードしてjupyterで動かすとか、python起動して動かすとかしてください。
 
ゲームルール
　爆弾を使って蠢く敵を倒す。
　最初に持っている爆弾の数（同時設置数）は1個。
　最初の爆弾威力は隣のマスまで。
　アイテムを拾うと爆弾の数と威力が増減する。
　全ての敵を倒してから扉をくぐるとステージクリアー。
 
操作方法
　移動　ー　カーソルキー
　爆弾設置　ー　スペースキー
